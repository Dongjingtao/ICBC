## 19.

现在有两个双精度浮点型的数组如下

ouble xs[]={1,9,3,5,6};

double ys[]={10,11,4,7,13};

又有一个长方形的数组 Rectangle rects[]= new Rectangle[25];

要求：用 Rectangle的构造方法给rects数组的每个元素赋值,长和宽分别为xs和ys的元素；然后按照面积由小到大给 rects排序。请编写它的程序。



## 20.

## 22.
